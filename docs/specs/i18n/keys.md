# i18n Keys
## 命名
UI.<AREA>.<COMPONENT>.<ACTION>
- AREA: MENU / DIALOG / PANEL / BUTTON / INPUT / ERROR / ONBOARDING / SETTINGS / NEST / EXPORT / LIBRARY
- COMPONENT/ACTION: 意味で切る（文章で切らない）

## 例（主要導線）
- UI.MENU.FILE.OPEN
- UI.MENU.FILE.SAVE
- UI.MENU.FILE.SAVE_AS
- UI.MENU.FILE.RECENT
- UI.BUTTON.EXPORT
- UI.BUTTON.NEST
- UI.DIALOG.SUPPORT.TITLE
- UI.PANEL.ERROR.TITLE
- UI.PANEL.ERROR.DETAIL
- UI.SETTINGS.LANGUAGE
- UI.SETTINGS.UNIT_SYSTEM

## SHORT/LONG
レイアウト制約が厳しい場合のみ、.SHORT / .LONG を許可する（乱用禁止）。
