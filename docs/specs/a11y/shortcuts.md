# Shortcuts (SSOT)
## Global（全モード）
- Ctrl/Cmd+O : Open
- Ctrl/Cmd+S : Save
- Ctrl/Cmd+Z : Undo
- Shift+Ctrl/Cmd+Z : Redo
- Esc : Cancel / return to safe mode
- Ctrl/Cmd+F : Library search
- F1 : Help/Support

## View
- Arrow keys : Pan (keyboard)
- + / - : Zoom in/out

## CAD (tool hotkeys)
- L : Line
- C : Circle
- A : Arc
- D : Dimension
- T : Annotation/Text

## Nest/Export
- N : Nest
- E : Export

### Notes
- 1キー系は「テキスト入力中は無効」などモード依存を明示する
- 重複は禁止（同一モード内で同一キー割当は不可）
